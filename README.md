# Car Dealer AGNT

Premium single-page marketing website for Car Dealer AGNT — a dealer operating system with AI listings, multi-platform publishing, and lead intelligence.

## Quick Start

```bash
npm install
npm run dev
```

Opens at [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- **Vite** — fast build tooling
- **React 19** — component architecture
- **Custom CSS** — precision design tokens, no framework bloat
- **Inter** — typography via Google Fonts

## Structure

```
src/
  components/    # All page sections + modal + sticky bar
  hooks/         # useReveal (scroll animation) + useScrolled
  index.css      # Design tokens + global styles
  App.jsx        # Main layout
public/
  agnt-logo.png          # AGNT brand mark
  hero-car-schematic.png # Bronze schematic hero visual
```
